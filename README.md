# AW_Bistro
Proyecto de Aplicaciones Web: 

**Plataforma Web de Gestión para Restaurantes (AW Bistro) | PHP, JavaScript (AJAX), MySQL, CSS , HTML**

- Desarrollo full-stack de una aplicación web orientada a objetos desde cero, implementando una arquitectura modular para la gestión segura de formularios, sesiones y conexión a base de datos.  
- Diseño de un sistema de Control de Acceso Basado en Roles (RBAC), creando interfaces y lógicas de negocio específicas para Clientes, Administradores, Camareros y Cocineros. 
- Implementación de un catálogo dinámico con carrito de la compra e historial de pedidos, integrando peticiones asíncronas (AJAX) para la gestión en tiempo real de valoraciones y productos favoritos.
